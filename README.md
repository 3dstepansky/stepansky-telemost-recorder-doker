# Stepansky Telemost Recorder (Docker Edition) v0.001

Автономный рекордер для Яндекс.Телемост, работающий в Docker-контейнере. Позволяет записывать встречи без установки дополнительных аудио-драйверов и без платной подписки (вход в режиме гостя).

## 🚀 Основные достижения (v0.001)

В первой версии (MVP) реализованы следующие решения:
1.  **Zero-Cost Capture**: Запись аудио напрямую из WebRTC потока (Puppeteer), что исключает необходимость в виртуальных аудио-кабелях.
2.  **Smart Guest Mode**: Бот заходит на встречу автоматически, не требуя авторизации.
3.  **Clean Naming Fix**: Решена проблема навязчивого префикса "Гость". Бот заходит под чистым именем (например, "Бот-Ассистент") через прямую инъекцию в DOM-прототипы.
4.  **Headless-стабильность**: Бот успешно работает в Docker (Linux) в полностью фоновом режиме без графического интерфейса.
5.  **Auto-Cleanup**: Использование ключа `--rm` в Docker гарантирует очистку ресурсов после завершения встречи.

## 🛠 Требования
- Docker
- Docker Compose

## 📦 Быстрый старт

1.  Клонируйте репозиторий.
2.  Создайте файл `.env` (используйте `.env.example` как шаблон).
3.  Запустите запись одной командой:
    ```bash
    docker-compose run --rm recorder "ВАША_ССЫЛКА_НА_ТЕЛЕМОСТ"
    ```

## 📁 Структура
- `recorder.js` — ядро захвата и обхода интерфейса Яндекса.
- `Dockerfile` — окружение с предустановленным Chromium и зависимостями.
- `recordings/` — папка, куда сохраняются ваши `.webm` файлы.
